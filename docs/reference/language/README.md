# Language Reference

2. [expr.md](expr.md): 表达式、调用、运算符和字面量。
3. [type.md](type.md): 类型写法、指针、数组、tuple 和函数指针。

- [pointer.md](pointer.md): 显式指针、取地址、解引用和 `T[*]`。
- [ref.md](ref.md): 显式 `ref` 绑定与 `ref` 参数。
